# 🚀 OptiMatch — AI-Powered Resume Analyzer & ATS Scoring System

> 💡 Not just a resume checker — an AI-driven hiring intelligence engine.

---

## 🧠 Overview

OptiMatch is an advanced **AI-powered ATS (Applicant Tracking System)** that evaluates resumes against job descriptions using **semantic analysis, NLP, and intelligent scoring**.

Upload your resume, paste a job description, and get:

- 🎯 ATS Match Score  
- 🔍 Skill Gap Analysis  
- 🧠 AI-Based Insights  
- 📊 Score Breakdown  
- ⚡ Real-Time Evaluation  

---

## ✨ Features

### 📄 Resume Parsing
- Extracts text from PDF resumes  
- Handles real-world formatting  
- Converts unstructured data into insights  

---

### 🎯 ATS Scoring Engine
- Match score (0–100)  
- Keyword + semantic matching  
- Identifies:
  - ✅ Matched skills  
  - ❌ Missing skills  
  - ⚠️ Improvement areas

## 🚀 Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/bunny182005/AI-Powered-Resume-Analyzer-ATS-Scoring-Job-Fit-Matching-System
cd AI-Powered-Resume-Analyzer-ATS-Scoring-Job-Fit-Matching-System
2️⃣ Backend Setup
cd backend
python -m venv .venv
source .venv/bin/activate   # Mac/Linux
pip install -r requirements.txt
PYTHONPATH=. uvicorn src.main:app --reload
3️⃣ Frontend Setup
cd frontend

npm run dev
